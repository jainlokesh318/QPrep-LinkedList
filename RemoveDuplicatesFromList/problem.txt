Problem Description
Given a sorted linked list, delete all nodes that have duplicate numbers, leaving only distinct numbers from the original list.
Return the linked list sorted as well.

Input format:
First line contains N, number of elements in the linked list.
Next line contains N space separated integers, elements of the linked list.

Output format
Print the linked list after removing duplicates.

Constraints
0 <= Number of nodes <= 10^5
-10^9 <= ListNode.val <= 10^9

Sample Input 1
5
1 2 2 3 3

Sample Output 1
1

Explanation 1
Node 2 and 3 have 2 occurrences each. Node 1 is distinct

Sample Input 2
3
1 3 4

Sample Output 2
1 3 4

Explanation 1
There are no duplicates present in the given linked list
